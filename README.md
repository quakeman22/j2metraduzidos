# 📝 J2ME Traduzidos

Esse repositório é um espaço dedicado a jogos Java (J2ME) traduzidos para português.

A ideia aqui é simples: preservar jogos antigos de celular e deixar eles mais acessíveis pra quem fala PT-BR. Se você já jogou em Nokia, Sony Ericsson ou similares, vai se sentir em casa.

---

## 🛠️ Como usar

Você pode rodar os jogos usando um emulador J2ME.

### 📱 Android
Recomendado: **J2ME Loader e JL-MOD**

Passos:
1. Baixe o jogo aqui do repositório
2. Abra o emulador
3. Selecione o arquivo `.jar`
4. Aproveite o jogo 👍
